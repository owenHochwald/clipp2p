# ClipP2P

A peer-to-peer universal clipboard for your terminal.
Built with Go, libp2p, and Bubble Tea.

## Features
- **Zero Config:** Automatically finds peers on your local network using mDNS.
- **TUI Dashboard:** See connection status and sync history in your terminal.
- **E2E Encrypted:** Data is encrypted between nodes.

## Tech Stack
- **Go** (Golang)
- **libp2p** (P2P Networking)
- **Bubble Tea** (TUI Framework)
- **golang.design/x/clipboard** (Native Clipboard)
